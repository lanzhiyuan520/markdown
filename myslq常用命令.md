#### myslq常用命令

1.查看所有数据库

```mysql
show databases;
```

2. 查看所有表

```mysql
show tables;
```

3. 查看表结构

```mysql
desc table_name;
```

4. 切换数据库

```mysql
use 库名;
```

5. 查看表所有数据

```mysql
select * from table_name;
```

6、创建数据库

```mysql
create database test;
```

7.清空表数据

```mysql
truncate test;
```

8.mysql端口查询

```mysql
sudo netstat -anp | grep mysql
```
